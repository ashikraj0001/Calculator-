<p><b>The Tool Installation Process Is Very Easy.. Just Open Your Termux & Type This Provided Commands!!</b></p>

- $ `apt update && apt upgrade`
- $ `pkg install python`
- $ `pkg install python2`
- $ `pkg install git`
- $ `pkg install figlet`
- $ `git clone https://www.github.com/ashik0001/Calculator-.git`
- $ `cd Calculator-`
- $ `chmod +x L12.py`
- $ `python L12.py`
